# OpenMP Tutorial


## Build from source

Build Enviorment:
- Make 3.8.1 
- gcc  10.2.0

Checkout to project root and run following commands:
```shell
# if build on linux
$ make

# if build on MacOS
$ make -e CXX=[appropriate_gcc_complier]
```
